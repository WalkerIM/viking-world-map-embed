# Viking World Map — embeddable build

Static, iframe-friendly build of the [Interactive Viking World Map](https://vikingsbrand.co/pages/viking-world-map) by VikingsBrand
(45 sourced locations, 6 guided journeys, era slider 793–1066 AD, voyage measure tool).

Embed it anywhere:

```html
<iframe src="https://walkerim.github.io/viking-world-map-embed/" width="100%" height="640"
  style="border:0;border-radius:12px" loading="lazy" title="Interactive Viking World Map, 793–1066 AD"></iframe>
<p>Map: <a href="https://vikingsbrand.co/pages/viking-world-map">Interactive Viking World Map by VikingsBrand</a></p>
```

Free for teachers, writers and history sites with the attribution link above.
